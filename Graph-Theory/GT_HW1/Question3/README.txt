To test the script you need to download the dataset and put the dataset into the same folder with the script for easier execution.
Dataset download link: http://snap.stanford.edu/data/soc-Epinions1.html

Execute the following command to test the script:
python Question3.py --path "your_relative_path_to_soc-Epinions1.txt_file"

Requirements:
- matplotlib.pyplot 
- snap
- scipy.io 
- numpy 
- argparse
